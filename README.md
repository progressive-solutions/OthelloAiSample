# ♟️ OthelloAiSample (オセロ AI 思考エンジンサンプル)

## 概要

`OthelloAiSample`は、外部のオセロ対戦管理プログラム（ジャッジ）と連携するために設計された AI 思考エンジンのサンプルプロジェクトです。新人研修の題材として、このプロジェクトのコアロジックを改良し、より強力なオセロ AI を開発することを目的としています。

このサンプルは、**標準入出力（stdin/stdout）**を通じたシンプルな通信プロトコルに従って動作します。

---

## 📂 プロジェクト構成

本プロジェクトは、責任を分離するために 3 つの Java ファイルに分割されています。

| ファイル名            | 役割                       | 編集推奨度           |
| :-------------------- | :------------------------- | :------------------- |
| `AIEntry.java`        | **エントリポイント＆通信** | **変更不要**         |
| `Board.java`          | **ゲームルール＆盤面管理** | **変更不要**         |
| `ThinkingEngine.java` | **思考ロジックの核**       | **メインの編集対象** |

### 役割の詳細

- **`AIEntry.java`**: ジャッジプログラムからの標準入力を受け取り、プロトコル（`COLOR`, `MOVE BOARD:`）を解析し、`ThinkingEngine`を呼び出します。
- **`Board.java`**: AI が内部で盤面を操作・評価するために必要な、オセロのルール（合法手判定、石のカウントなど）を提供します。
- **`ThinkingEngine.java`**: 実際に着手を決定するアルゴリズムを記述する場所です。初期状態では**ランダムな合法手**を返す最弱 AI が実装されています。

---

## 🛠️ ビルドと実行

このプロジェクトを外部ジャッジ（例: `OthelloJudgeConsole`）と対戦させるには、実行可能な JAR ファイルを作成する必要があります。

### 1. Java ファイルのコンパイル、実行可能な JAR ファイル生成

すべての`.java`ファイルをコンパイルします。

```bash
mkdir -p classes
javac -d classes  -source 21 -target 21 *.java; jar cvfm SampleAI.jar Manifest.txt -C classes .

```

例として、SampleAI.jar という名前で JAR ファイルを生成するコマンドを記載します。
jar ファイル名はお好きな名前を指定してください。

### 2. 単体実行テスト

作成した JAR ファイルは、ジャッジプログラムに提供する前に、以下のコマンドで起動できるか確認してください。

```
java -jar SampleAI.jar
```

起動したプログラムはジャッジプログラムからの入力待ちの状態となります。
例として、初手時は下記となります。

```
COLOR 1
MOVE BOARD:0000000000000000000000000002100000012000000000000000000000000000
```
